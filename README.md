# my-vim
## Manual installation
1. copy files to your home directory:

        git clone git://github.com/k19953014/my-vim.git

2. rename it to `.vim` also be fine:

        mv my-vim .vim

3. link the vimrc to

        ln -s .vim/vimrc .vimrc

## Notice

1. remove `bundle`

2. bundle needs Plugin(type in the vim)

        :PluginInstall
